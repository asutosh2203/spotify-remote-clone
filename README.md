# Remote Spotify Clone
A controller for your spotify app.

**You will need to have a premium account with Spotify to be able to use this app.**

This app uses a wrapper around the spotify web playback sdk called spotify spotify-web-api-js. You can view your playlists and control play, pause and changing of songs from the console within the app. 

By default it show the "Top 100 Rock songs on Spotify" but you can it by changing the ID in `spotify.getPlaylist()` in App.js. 

## To start with the app:
Clone the repository and navigate into the root directory. 

Run `npm i` to install the required dependencies locally onto your device. 

Run `npm start` to spin up the development mode.

Browse to http://localhost:3000 to view the project.
